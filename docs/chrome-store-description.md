### **Permission Justifications**

**Storage Permission:**
```
Required to save user preferences, profile settings, trigger category selections, warning cache, and customization options locally on your device. No data is sent to external servers.
```

**Tabs Permission:**
```
Required to detect which streaming platform you're currently viewing and to inject the warning banner into the appropriate video player interface.
```

**ActiveTab Permission:**
```
Required to access the current tab's video player, detect media playback, analyze subtitles in real-time, and display warnings at the correct timing during video playback.
```

**Alarms Permission:**
```
Required to schedule periodic warning database updates and manage timed warning notifications based on video playback position.
```

**Host Permissions (Netflix, YouTube, Prime Video, Hulu, Disney+, Max, Peacock, Supabase):**
```
Required to:
1. Detect video playback on supported streaming platforms
2. Inject warning banner overlay into video player interfaces
3. Access subtitle/caption data for real-time trigger detection
4. Query the Supabase database for community-submitted warnings (anonymous video ID lookups only)

No personal data, viewing history, or video content is collected. Only anonymous video IDs are sent to the database to retrieve relevant warnings.
```
