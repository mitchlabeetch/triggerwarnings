## Getting Started

### First Time Setup

After installing the extension:

1. **Click the extension icon** in your browser toolbar
2. **Choose your trigger categories**: Select what you want to be warned about
3. **Set your warning style**: Decide how you want to be notified
4. **Start watching**: Navigate to any supported streaming platform

That's it! The extension will now monitor your viewing and alert you when triggers appear.

**üí° Tip**: You can enable/disable categories anytime without losing your settings.

### Warning Display Styles

#### Banner Style (Default)
- **Description**: Full-width banner at top of video
- **Visibility**: High (hard to miss)
- **Best for**: First-time viewers, high-sensitivity triggers

#### Indicator Style
- **Description**: Small icon in corner of video
- **Visibility**: Low-profile
- **Best for**: Re-watches, less sensitive content

#### Notification Style
- **Description**: Browser notification (requires permission)
- **Visibility**: Outside the video player
- **Best for**: Multi-tasking, background viewing

### Advanced Settings

#### Pre-Watch Summary
- **What it does**: Shows all warnings before you start
- **Use case**: Decide if you want to watch before investing time
- **Toggle**: Settings ‚Üí "Show pre-watch summary"

#### Warning Lead Time
- **What it does**: How many seconds before the trigger to warn you
- **Options**: 3s, 5s, 10s, 15s
- **Recommended**: 5-10 seconds (time to look away or skip)

#### Auto-Skip
- **What it does**: Automatically skip triggering scenes
- **Warning**: May miss plot-relevant content
- **Toggle**: Settings ‚Üí "Auto-skip triggers"

## Using the Extension

### On Netflix

1. **Navigate to any show/movie**
2. **Look for the extension icon**: 
   - Green checkmark = Warnings available
   - Yellow question mark = No warnings yet (be the first to add!)
   - Gray = Not yet analyzed
3. **Click "View Warnings"** to see the pre-watch summary
4. **Start watching**: Warnings will appear in real-time

### On YouTube

1. **Open any video**
2. **Extension activates automatically** for videos with warnings
3. **Look for the indicator** next to the video title
4. **Warnings appear** as you watch

### On Other Platforms

The extension works similarly across:
- **Prime Video** (including IMDb TV)
- **Hulu** (ads may briefly hide warnings)
- **Disney+** (family-friendly, fewer warnings)
- **Max (HBO Max)** (mature content library)
- **Peacock** (NBC content)

## Managing Profiles

### What Are Profiles?

Profiles let you switch between different sensitivity settings instantly. For example:

- **"Solo Viewing"**: All triggers enabled
- **"With Kids"**: Only high-sensitivity triggers
- **"Work-Safe"**: Maximum warnings (violence, language, etc.)

### Creating a Profile

1. Open **Settings**
2. Click **"Profiles"** tab
3. Click **"+ New Profile"**
4. Name your profile (e.g., "Weekend Viewing")
5. Select trigger categories
6. Click **"Save"**

### Switching Profiles

**Quick Switch**: Click the extension icon ‚Üí Profile dropdown ‚Üí Select profile

**Keyboard Shortcut**: Press `Alt+Shift+P` (customizable in browser settings)

### Sharing Profiles

You can export/import profiles:

1. **Export**: Settings ‚Üí Profiles ‚Üí "Export" ‚Üí Save JSON file
2. **Share**: Send the JSON file to friends/family
3. **Import**: Settings ‚Üí Profiles ‚Üí "Import" ‚Üí Select JSON file

## Contributing Warnings

### Adding a Warning

Help the community by adding warnings for content you've watched:

1. **While watching**: Note the timestamp when a trigger occurs
2. **Pause the video**
3. **Click the extension icon**
4. **Click "Add Warning"**
5. **Fill out the form**:
   - **Trigger type**: Select from categories
   - **Timestamp**: Auto-filled (or manually adjust)
   - **Duration**: How long the trigger lasts
   - **Severity**: Mild, Moderate, Severe
   - **Description** (optional): Brief context (no spoilers!)
6. **Submit**

### Warning Guidelines

**DO:**
- ‚úÖ Be specific (e.g., "Spider closeup" not just "Spider")
- ‚úÖ Include duration (5 seconds vs. 2 minutes matters)
- ‚úÖ Rate severity honestly
- ‚úÖ Use timestamps from the video player (not your memory)

**DON'T:**
- ‚ùå Add joke/fake warnings (instant removal + ban)
- ‚ùå Include spoilers in descriptions
- ‚ùå Exaggerate severity
- ‚ùå Be vague (e.g., "Something bad happens")

### Editing Your Warnings

Made a mistake? You can edit your submissions:

1. Click the extension icon ‚Üí "My Warnings"
2. Find the warning
3. Click "Edit"
4. Update details
5. Save

## Voting on Warnings

### Why Vote?

Community voting ensures warning accuracy:
- **Upvotes** = "This warning is accurate and helpful"
- **Downvotes** = "This warning is wrong, misleading, or spam"

Warnings with too many downvotes are automatically hidden.

### How to Vote

**During Playback**: When a warning appears, you'll see ‚¨ÜÔ∏è and ‚¨ÜÔ∏è buttons

**After Watching**: 
1. Click extension icon ‚Üí "Review Warnings"
2. See all warnings for current content
3. Vote on each

### Voting Guidelines

**Upvote if:**
- ‚úÖ Warning is accurate (timestamp, severity, type)
- ‚úÖ Description is helpful (clear, no spoilers)
- ‚úÖ You found it useful

**Downvote if:**
- ‚ùå Wrong timestamp (off by >5 seconds)
- ‚ùå Wrong trigger type
- ‚ùå Severity is way off
- ‚ùå Contains spoilers
- ‚ùå Spam or joke warning

**Don't vote if:**
- ‚ö†Ô∏è You disagree with severity (subjective)
- ‚ö†Ô∏è You wouldn't personally need this warning (others might!)

## Troubleshooting

### Extension Not Working

**Check 1: Extension is enabled**
- Go to `chrome://extensions/`
- Find "Trigger Warnings"
- Toggle should be **ON** (blue)

**Check 2: You're on a supported site**
- Netflix, Prime Video, Hulu, Disney+, Max, Peacock, YouTube
- Other sites are not yet supported

**Check 3: Refresh the page**
- Press `Ctrl+R` (Windows/Linux) or `Cmd+R` (Mac)
- Extension injects on page load

**Check 4: Update the extension**
- Go to `chrome://extensions/`
- Click "Update" at the top
- Restart your browser

### Warnings Not Showing

**Possible reasons:**

1. **No warnings exist yet**: Be the first to add them!
2. **Profile mismatch**: Check if the trigger category is enabled in your current profile
3. **Content not identified**: Some content doesn't have IMDb IDs yet (working on this)
4. **Extension popup is open**: Close it and warnings should reappear

### Performance Issues

If videos are lagging or buffering:

1. **Disable AI detection**: Settings ‚Üí "Local Detection" ‚Üí OFF
2. **Reduce warning frequency**: Settings ‚Üí "Max warnings per minute" ‚Üí Lower value
3. **Clear cache**: Settings ‚Üí "Advanced" ‚Üí "Clear warning cache"
4. **Restart browser**: Close and reopen to free up memory

### Reporting Bugs

If you encounter a bug:

1. **Open the console**: Press `F12` ‚Üí "Console" tab
2. **Look for errors**: Red text indicates issues
3. **Take a screenshot**
4. **Report**: [GitHub Issues](https://github.com/mitchlabeetch/Trigger_Warnings/issues)

Include:
- Browser version
- Extension version
- Steps to reproduce
- Screenshots/console errors

## FAQ

### General Questions

**Q: Is this extension free?**  
A: Yes! Forever. No ads, no paid tiers, no tracking.

**Q: Does it work on mobile?**  
A: Not yet. Browser extensions on mobile are limited. We're exploring alternatives.

**Q: Can I use it on smart TVs?**  
A: Unfortunately, no. Extensions only work in web browsers on desktops/laptops.

**Q: What streaming services are supported?**  
A: Netflix, Prime Video, Hulu, Disney+, Max, Peacock, and YouTube. More coming soon!

### Privacy Questions

**Q: Do you track what I watch?**  
A: **No.** Your viewing history never leaves your device.

**Q: What data do you collect?**  
A: Only anonymous warning data (timestamps, votes). No personal info.

**Q: Can other users see my profile?**  
A: No. Profiles are stored locally on your device.

**Q: Is my data encrypted?**  
A: Yes. All API communication uses HTTPS. Local data is encrypted by your browser.

### Technical Questions

**Q: Why does it need so many permissions?**  
A: We explain each permission in [SECURITY.md](../SECURITY.md). TL;DR: We only use what's necessary.

**Q: Can I use it with a VPN?**  
A: Yes. VPNs don't affect the extension.

**Q: Does it work with ad blockers?**  
A: Yes, but if you block our API domain, warnings won't load.

**Q: How much RAM/CPU does it use?**  
A: Minimal. ~10-20MB RAM. AI detection adds ~50-100MB when active.

### Contributing Questions

**Q: How do I add support for a new platform?**  
A: See [docs/PROVIDERS.md](PROVIDERS.md) for a developer guide.

**Q: Can I translate the extension?**

**Q: I found a security issue. What do I do?**
